TypeScript is a typed superset of JavaScript created by Microsoft. It compiles to plain JavaScript in any browser and has many syntactic similarities to JavaScript while also serving to emulate traditional Object Oriented Languages like C# or Java. Essentially, anything that's in JavaScript is part of TypeScript.

A programming language that contains all the features of a given language and has been expanded or enhanced to include other features as well.
